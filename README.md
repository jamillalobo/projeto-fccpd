# Projeto 2 

O presente projeto possui 5 desafios realizados para a disciplina de Fundamentos da Computação Concorrente, Paralela e distriuída. Os desafios propostos foram os seguintes:

1. Criar dois containers que se comunicam por uma rede Docker customizada.
2. Demonstrar persistência de dados usando volumes Docker.
3. Usar Docker Compose para orquestrar múltiplos serviços dependentes.
4. Criar dois microsserviços independentes que se comunicam via HTTP.
5. Criar uma arquitetura com API Gateway centralizando o acesso a dois microsserviços.
